HTML

  • HyperText Markup Language, or HTML, is the most basic building block of websites - this includes text, images, or parts of a page’s layout. It handles the content of a website, as well as its structure and the meaning of each element on the page.
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • Cascading Style Sheets, or CSS, is the language used to describe the presentation of a website that’s created with HTML and how each element on a webpage is rendered. It is used to add styling to contents created using HTML, these includes fonts, colours and spacing.

Git

  • Git is a free, open source version control system that allows web developers to collaborate on projects without overwriting each other's work. It also allows developers to revert to previous versions of the code if a problem is discovered.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a lightweight programming language that is used to provide logic and interactivity for websites.
